# IncliArray - NumPy like library for C++

## Local build

1. Clone this repo, and change directory to it.
    ```sh
    git clone https://github.com/inclinedadarsh/incliarray.git
    cd incliarray
    ```

2. Type the following commands:

    ```sh
    mkdir build
    cd build
    cmake ..
    make
    ./IncliArray
    ```

    > Make sure CMake is installed
